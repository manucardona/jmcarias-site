Poverty Mapping in Panama

🚀 What it does
A tool to classify household poverty in Panama using logistic regression, satellite radiance data (VIIRS), and five household survey questions. It includes an explainable boosting machine (EBM) model and a spatial visualization dashboard.

🎯 Why I made it
To explore how small, privacy-safe data inputs can power accurate poverty predictions — especially where traditional data collection is limited.

🛠 Tools & Technologies
Python (scikit-learn, interpretml), ChromaDB, Folium, Flask, Jupyter Notebooks.

📚 What I learned

  • Working with low-resolution VIIRS data and MOSAIKS features
  • Deploying interpretable ML (EBMs)
  • Building full pipelines from model training to deployment

🌍 Real-world use / impact
Used to demonstrate a low-cost poverty targeting prototype during a university presentation on data-driven development.

👥 Team
Solo project, though inspired by work with Microsoft AI for Good and IPA.
